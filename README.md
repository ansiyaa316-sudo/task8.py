# File Handling(TXT& CSV)

This Python program demonstrates basic file handling and CSV file operations.
It allows the user to:

Store personal data (name, age, email) in a text file

Read and display data from the file

Handle file errors safely

Create, write, and read data from a CSV file

This project is ideal for beginners learning Python file operations.

## Text File Operations (data.txt)

Create a text file

Write user details (name and age)

Read and display file contents

Append email information

Use exception handling to manage missing files

Read files using both open() and with statement

## CSV File Operations (students.csv)

Create a CSV file

Write student records (Name, Age, Branch)

Read and display CSV data row by row

## Files Used

data.txt – Stores user information

students.csv – Stores student records in tabular form

## Concepts Covered

File modes: r, w, a

Reading and writing text files

Exception handling (try-except)

CSV module usage

Context manager (with open())

## Run the program:

python file_handling.py
